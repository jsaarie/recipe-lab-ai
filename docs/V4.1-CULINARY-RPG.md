# v4.1 — Culinary RPG & Gamification

## Overview

Transform Recipe Lab AI into a gamified cooking experience. Users earn Skill XP for cooking activities, level up through culinary title tiers, unlock milestone badges, and discover new recipes via an AI-powered Random Recipe Roll.

## XP System

### Actions & Point Values

| Action | XP | Trigger |
|--------|----|---------|
| Complete a recipe | 100 | Finish the last step in Lab View |
| Rate a recipe | 15 | Submit a star rating (v4.0 feedback) |
| Add cook notes | 20 | Submit cook notes (v4.0 feedback) |
| OCR scan | 50 | Successfully scan a recipe via OCR |

- XP is cumulative and never lost.
- Each action awards XP once per recipe instance (e.g., updating a rating does not re-award XP, but rating a different recipe does).

### Culinary Title Tiers

7 tiers with escalating XP thresholds:

| Tier | Title | XP Required |
|------|-------|-------------|
| 1 | Home Cook | 0 |
| 2 | Prep Cook | 200 |
| 3 | Line Cook | 500 |
| 4 | Sous Chef | 1,200 |
| 5 | Head Chef | 2,500 |
| 6 | Executive Chef | 5,000 |
| 7 | Master Chef | 10,000 |

### Display

- **Profile Page** — Full XP bar showing current XP, progress to next tier, current title, and badge collection.
- **Nav Bar Badge** — Compact badge showing current culinary title.
- **XP Updates** — Calculated on page load. No real-time toasts.

## Milestone Badges

Unlockable badges for reaching specific milestones. Displayed on the user's profile.

### Core Badges

| Badge | Condition |
|-------|-----------|
| First Flame | Complete your first recipe |
| Ten Timer | Complete 10 recipes |
| Quarter Century | Complete 25 recipes |
| Critic's Voice | Submit your first rating |
| Field Notes | Add your first cook notes |
| Gold Standard | Rate 3 recipes 5 stars in a row |

### Extended Badges

| Badge | Condition |
|-------|-----------|
| OCR Pioneer | Scan your first recipe via OCR |
| Cuisine Explorer | Cook recipes from 5 different cuisines |
| Speed Cook | Complete a recipe in under 15 minutes |
| Weekend Warrior | Complete 3 recipes in a single weekend |
| Bookworm | Save 20 recipes to your library |
| Night Owl | Complete a recipe after 10 PM |
| Early Bird | Complete a recipe before 8 AM |

Badge count and icons are shown on the profile page. New badge unlocks are surfaced when the user next visits their profile.

## Random Recipe Roll

### Concept

An AI-powered "Surprise Me" feature that generates a new recipe recommendation based on the user's taste profile.

### How It Works

1. User taps "Recipe Roll" (accessible from Library or home).
2. A cookbook page-turn animation plays.
3. AI analyzes the user's past ratings and inferred cuisine/ingredient preferences.
4. A brand-new recipe is generated and presented — not from the user's library, but a fresh suggestion.
5. User can save the suggested recipe to their library or roll again.

### AI Inputs

- Star ratings from v4.0 feedback
- Cuisine tags inferred from saved recipes
- Ingredient frequency across saved recipes
- Avoidance signals (low-rated recipes, ingredients not used)

### UX

- **Page-Turn Animation** — Cookbook-style page flip to reveal the recipe suggestion.
- **Recipe Card** — Displays recipe name, cuisine, estimated time, and key ingredients.
- **Actions** — "Save to Library", "Cook Now", or "Roll Again".

## Data Model

### User XP Document

New fields on the user document or a dedicated `userProgress` collection:

```ts
interface UserProgress {
  userId: string
  totalXp: number
  currentTier: number          // 1–7
  currentTitle: string         // e.g., "Sous Chef"
  badges: string[]             // badge IDs unlocked
  xpLog: {                     // track per-recipe XP awards
    recipeId: string
    actions: string[]          // e.g., ["complete", "rate", "notes"]
    awardedAt: Date
  }[]
}
```

### API Routes

- `GET /api/user/progress` — Returns XP, tier, title, badges
- `POST /api/user/progress/xp` — Award XP for an action (server-validated)
- `GET /api/recipes/roll` — Generate an AI recipe suggestion based on user profile
